# Playtest Guide - Empire of Glass

## Overview
This guide helps playtesters provide valuable feedback on the core gameplay loop and game feel.

## Getting Started

### Installation
1. Download the build from [distribution link]
2. Extract to a folder
3. Run `EmpireOfGlass.exe` (Windows) or `EmpireOfGlass.app` (Mac)

### Controls
- **Arrow Keys / A/D**: Switch lanes in Swarm mode
- **Mouse Click**: Interact with UI elements
- **Number Keys 1-3**: Quick scene navigation (Boot scene only)
- **ESC**: Pause menu

## What We're Testing

### Priority 1: Core Loop Satisfaction
**Focus Areas:**
- Does the swarm multiplication feel satisfying?
- Is the math gate feedback clear and rewarding?
- Does the forced rotation (Swarm → City → Raid) feel natural?

**Questions to Consider:**
1. How satisfying is it to see your swarm multiply? (1-10)
2. Can you understand what each math gate does?
3. Does the game communicate your progress clearly?
4. Do you want to keep playing after each loop?

### Priority 2: Difficulty & Balance
**Focus Areas:**
- Is the game too easy or too hard?
- Do you feel progression over time?
- Are there frustrating moments?

**Questions to Consider:**
1. Did you die? If so, was it fair?
2. Did you feel challenged but not overwhelmed?
3. Did you want to retry after failing?
4. Are the rewards worth the effort?

### Priority 3: Visual Clarity
**Focus Areas:**
- Can you distinguish hero from enemies?
- Are interactive elements obvious?
- Is the UI readable?

**Questions to Consider:**
1. Could you tell what things do by looking at them?
2. Did any visual elements confuse you?
3. Is the text readable at your resolution?
4. Are colors distinct enough?

### Priority 4: Performance
**Technical Feedback:**
- Note any frame drops or stuttering
- Report any crashes or bugs
- Mention your system specs if issues occur
